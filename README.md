# Verilog Fundamentals

All code typed manually. No copy-paste.

## Objective

Build Verilog syntax fluency from zero through disciplined daily practice.  
Foundation for RTL design and microarchitecture work targeting Tier-1 semiconductor firms.

## Learning Path

| Stage | Source | Focus |
|---|---|---|
| 1 | chipverify.com | Syntax, data types, operators, modules |
| 2 | HDLBits — Basics | Wire, NOT, AND, OR, XOR gates |
| 3 | HDLBits — Vectors | Multi-bit logic, concatenation, replication |
| 4 | HDLBits — Modules | Hierarchy, instantiation, port mapping |
